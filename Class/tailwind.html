@tailwindcss/typography

Beautiful typography for your Tailwind CSS projects.

The @tailwindcss/typography plugin adds a set of customizable prose classes that you can use to add beautiful typographic defaults to any vanilla HTML, like the output you'd get after parsing some Markdown, or content you pull from a CMS.

Typography Should be easy.

The @tailwindcss/typography plugin is a great way to style your prose content without adding any CSS. Just add the plugin to your project and use the prose classes to style your content.

The plugin includes a set of default styles that you can customize to match your brand. You can also use the typography classes to create custom styles for your content.
